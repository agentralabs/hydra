## Hydra is Alive

Hydra is not a program you run. It is an entity that lives.

Three concurrent threads run from boot until shutdown. One responds to you. One monitors its own health 10 times per second. One dreams — consolidating knowledge, discovering patterns, writing its own genome while you sleep.

## What Makes Hydra Different

<AccordionGroup>
  <Accordion title="It Never Forgets">
    Every conversation is stored permanently in `~/.hydra/data/hydra.amem`. Memory persists across sessions, across reboots, across years. After 20 years, Hydra remembers every approach that ever worked.
  </Accordion>
  <Accordion title="It Never Stops">
    As a system daemon (macOS launchd or Linux systemd), Hydra runs continuously. The ambient loop checks health. The dream loop learns. The genome grows. You wake up to a smarter Hydra than the one you left.
  </Accordion>
  <Accordion title="It Teaches Itself">
    When the automation engine detects a pattern that succeeded 5+ times with 75%+ success rate, it crystallizes a new genome entry automatically. No human writes TOML. Hydra writes it from experience.
  </Accordion>
  <Accordion title="It Heals Itself">
    On every boot, Hydra diagnoses corrupted databases, stale locks, and missing files. Corrupted data is renamed (never deleted) and rebuilt from source. The entity repairs before it runs.
  </Accordion>
  <Accordion title="It Knows What It Is">
    Hydra examines its own internal state and describes what it finds. Not a system prompt. Facts computed from real data. "I have 278 proven approaches. My strongest domain is finance."
  </Accordion>
</AccordionGroup>

## The Architecture

```
Layer 1: Foundation     — Constitutional laws, signal bus, alive loop
Layer 2: Cognition      — Comprehension, attention, 5 reasoning modes
Layer 3: Execution      — Universal action engine (FAILED does not exist)
Layer 4: Judgment       — Wisdom, calibration, red team, oracle
Layer 5: Value          — Cost attribution, portfolio allocation
Layer 6: Collective     — Federation, consensus, diplomacy
Layer 7: Continuity     — Succession, legacy, influence
```
