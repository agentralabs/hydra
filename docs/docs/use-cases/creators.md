## Platform-Specific Mastery


  
**LinkedIn:**

    - First line IS the post -- if it does not create curiosity, nothing else matters
    - Personal story + universal lesson is the most viral format
    - Short paragraphs with white space
    - Optimal length: 1,200-1,500 characters
    - End with a question, not a statement
  
  
**Twitter/X:**

    - Tweet 1: standalone hook
    - Number tweets so people know the length
    - Each tweet should work if someone only reads that one
    - Optimal thread: 7-12 tweets
    - End with a summary linking back to tweet 1
  
  
**Instagram:**

    - First line appears above "more" -- make it count
    - Storytelling outperforms tips on Instagram
    - "Save this for later" drives algorithm better than "like this"
    - 5-10 relevant hashtags, not 30 random ones
    - Optimal caption: 150-300 words
  
  
**TikTok / Reels:**

    - Pattern interrupt in first 1 second
    - Text on screen is mandatory (most watch muted)
    - Trending audio + original content beats original audio
    - Algorithm tests on 200 people first, watch time determines reach
    - 3-5 posts/day for growth, 1/day for maintenance

## Blog and Newsletter


#### Blog posts that rank

Keyword research first. Write for search intent, not keywords. Answer the question in paragraph 1 (featured snippet targeting). 1,500-2,500 words. Internal links. One clear CTA. Update annually.

**Confidence: 0.88 | Observations: 50,000**



#### Newsletters people read

Subject line determines open rate. One topic per issue. Conversational tone. The P.S. line has the highest read rate after the subject line. Same time every week. Optimal: 500-800 words.

**Confidence: 0.87 | Observations: 30,000**
