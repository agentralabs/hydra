---
title: "How Hydra Thinks"
description: "6-stage cognitive pipeline — 85% of processing without LLM calls."
---

Every input passes through 6 stages. Most resolve without calling an LLM.


  
**Step: Comprehend**

    Domain detection, primitive mapping, memory resonance. Zero LLM calls.
  
  
**Step: Analyze Language**

    Intent extraction, affect classification, depth analysis, hedge detection.
  
  
**Step: Build Context**

    Five windows: active, historical, predicted, gap, anomaly.
  
  
**Step: Allocate Attention**

    Score items by urgency and relevance. Budget based on intent × affect.
  
  
**Step: Reason**

    Five modes simultaneously: deductive, inductive, abductive, analogical, adversarial.
    If synthesis confidence ≥ 0.35 → zero-token resolution. No LLM needed.
  
  
**Step: Deliver**

    Write-ahead audit receipt. Settlement recording. Response to user.
