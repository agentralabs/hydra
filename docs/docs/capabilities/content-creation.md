## Carousel Mastery

```
Situation: "creating a LinkedIn or Instagram carousel"
Approach:  10 slides max -- slide 1: bold hook -- slides 3-8: one idea
           per slide -- slide 9: summary -- slide 10: CTA
Confidence: 0.91 | Observations: 80,000
```

Key carousel principles from the genome:

- **Max 30 words per slide** -- one sentence per idea
- **Dark background + white text** converts best on LinkedIn
- **Bold sans-serif, 40pt minimum** -- icons over photos for professional content
- **Hook slide determines everything** -- if slide 1 fails, nobody swipes

## Social Media by Platform


  
**LinkedIn:**

    First line IS the post. Personal story + universal lesson is the most viral format. Optimal length: 1,200-1,500 characters. Confidence: 0.88.
  
  
**Twitter/X:**

    Tweet 1: standalone hook. Number the tweets. Each tweet should work alone. Optimal thread: 7-12 tweets. Confidence: 0.87.
  
  
**Instagram:**

    First line appears above "more" -- make it count. Storytelling outperforms tips. "Save this for later" drives algorithm better than "like this." Confidence: 0.86.
  
  
**TikTok/Reels:**

    Pattern interrupt in first 1 second. Text on screen is mandatory. Trending audio + original content beats original audio. The algorithm tests on 200 people first. Confidence: 0.85.
