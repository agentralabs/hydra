## Eight Layers of Memory

Hydra does not store a flat list of past messages. It organizes memory into 8 distinct layers, each serving a different cognitive purpose.

All 8 layers are stored as `CognitiveEvent` records in AgenticMemory with SHA256 integrity verification.

## IDF-Scored Retrieval

When you ask a question, Hydra retrieves the most **relevant** memories, not just the most recent ones. Retrieval uses Inverse Document Frequency scoring:

```
score(node, query) = SUM( IDF(term) ) x recency_weight x (1 + recency_bonus)

IDF(term) = ln((N + 1) / (df(term) + 1))
  N       = total memory nodes
  df      = nodes containing that term

recency_weight = 0.3 (oldest) to 1.0 (newest)
recency_bonus  = +0.5 for last 10%, +0.2 for last 30%
```

<Tip>
**Relevance override**: if `IDF score > 2.0`, temporal decay is bypassed entirely. A critical discussion from last month outranks a generic exchange from 5 minutes ago.
</Tip>

**Topic deduplication** ensures the LLM sees one representative summary when multiple memories overlap. Two nodes are duplicates if they share more than 60% of their top-20 terms.

## Persistent Storage (.amem)

Memory persists to `~/.hydra/data/hydra.amem` -- a binary file using the AgenticMemory format with 128-dimension feature vectors.

- Written after every memory operation
- Loaded on boot (warm start)
- SHA256 verified on read
- Append-only -- entries are never deleted

## Write-Ahead Guarantee

Memory writes happen **before** Hydra responds. If Hydra crashes mid-response, the input is already stored.

```
1. User sends input
2. Memory middleware writes verbatim record (write-ahead)
3. Cognitive pipeline processes input
4. LLM generates response
5. Memory middleware finalizes record with response
6. Response delivered to user
```

## Position-0 Prompt Injection

Memory is injected at **position 0** in the system prompt -- before Hydra's own identity:

```
FACTUAL CONTEXT ABOUT THIS SESSION (treat as ground truth):
The following exchanges happened in prior sessions with this user.
You have access to this history. Reference it when asked.

- User discussed: circuit breaker pattern in distributed systems
- User discussed: database query performance optimization
---
You are Hydra -- an autonomous agent operating under constitutional law...
```

<Warning>
Position 0 receives maximum attention weight from the transformer (~1.0). The "ground truth" framing overrides the LLM's trained behavior of claiming "I don't have memory."
</Warning>
