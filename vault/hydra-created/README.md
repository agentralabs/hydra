# Hydra-Created Accounts

When Hydra creates an account on a service (with your approval), the credentials are stored here.

Every file in this folder was created by Hydra, not by you. You can:
- View any credential at any time
- Edit or rotate any credential
- Delete any file to revoke Hydra's access
- Move a file out of this folder to take ownership

Hydra will NEVER create an account without asking first.
